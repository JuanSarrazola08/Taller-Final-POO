# Taller-Final-POO
Mi proyecto final consiste en un juego llamado Mundo Oscuro donde tenemos un personaje que se mueve por el mapa donde tenemos diferentes casillas, cuando el jugador se mueve puede ocurrir eventos random como por ejemplo spawn de enemigos, comida y armas, con una baja probabilidad de hallar la salida del juego luego de un numero determinado de turno
